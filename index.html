Bước 1: Tạo Google Sheet
Sheet sẽ là kho lưu trữ chung của toàn bộ leads YTK Global.
1

Mở link này trong tab mới

https://sheets.newclick để mở
2

Đặt tên file là "YTK GLOBAL CRM"

Click vào "Untitled spreadsheet" → gõ YTK GLOBAL CRM → Enter
3

Đổi tên tab thành "Leads"

Chuột phải vào tab "Sheet1" → Rename → gõ Leads → Enter
Bước 2: Cài Apps Script
Apps Script là cầu nối giữa CRM và Google Sheets. Copy/paste, không cần hiểu code.
1

Mở Apps Script

Trong Google Sheet → Extensions (Tiện ích mở rộng) → Apps Script
2

Xóa code cũ, dán code mới vào

3

Lưu lại

Ctrl+S (hoặc Cmd+S trên Mac)
Bước 3: Deploy & kết nối
Deploy để Apps Script chạy được từ internet, sau đó dán URL vào đây.
1

Nhấn Deploy → New deployment (Triển khai mới)

2

Cài đặt

• Type / Loại: Web app / Ứng dụng web
• Execute as: Me / Tôi
• Who has access: Anyone / Mọi người
→ Nhấn Deploy
3

Cấp quyền (Authorize)

Nếu thấy cảnh báo → Advanced / Nâng cao → Go to YTK... → Allow / Cho phép
4

Dán URL vào đây

URL dạng: https://script.google.com/macros/s/.../exec
